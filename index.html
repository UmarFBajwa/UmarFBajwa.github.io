THE KEYS TO BUILDING GREAT PRODUCT

Common Questions

A. What Makes a Great Product?

  • Fulfills a real need
  • Focuses on the User
  • Uses the correct technology to deliver the solution
  • Anticipates and solves problems
  • Works better than expected
  • Is aesthetically pleasing
  • Provides responsive, appropriate feedback
  • Becomes a necessity

B. How Often Should You Release?

  • As soon as possible
  • Use the MVP approach
  • Lean is not just a methodology but daily habit

C. What is the Most Important Rule?

  1. It's all about the User
  2. Only about the User
  3. Did I mention the User?!?

Quora

Quora is a good site both answering the question and from a html perspective. It lists each answer appropriately. It has links to related sources that are easy to read and click. It posts stats about the question that update automatically as Users add and comment. It lists subject categories at the top of the page. I see lots of content mixed in large paragraphs and unfamiliar code but it comes out very nicely. Good use of headers and paragraphs and links. It has solid answers from experts, links to excellent resources and more.
